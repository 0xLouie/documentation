## Verification Levels

Access also depends on verification:

<AccordionGroup>
  <Accordion title="Unverified" icon="circle">
    Limited functionality:
    - Browse markets only
    - Cannot enter predictions
    - Cannot challenge
    - Must complete email verification
  </Accordion>
  <Accordion title="Email Verified" icon="envelope">
    Standard access:
    - All Free tier features
    - Can enter predictions
    - Can challenge friends
    - Standard limits apply
  </Accordion>
  <Accordion title="Phone Verified" icon="mobile">
    Enhanced access:
    - Higher position limits
    - Competitive league eligibility
    - Reduced fraud checks
    - Trust score boost
  </Accordion>
  <Accordion title="ID Verified" icon="id-badge">
    Full access:
    - Maximum limits
    - Cash prize leagues
    - Creator program eligibility
    - Premium features
  </Accordion>
</AccordionGroup>

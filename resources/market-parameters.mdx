## AMM Parameters

The Automated Market Maker uses the following configuration:

| Parameter | Value | Description |
|-----------|-------|-------------|
| **Initial Liquidity** | 10,000 tokens | Starting pool for new markets |
| **Fee** | 2% | Platform fee on winning predictions |
| **Min Odds** | 1% | Lowest possible probability |
| **Max Odds** | 99% | Highest possible probability |
| **Slippage Tolerance** | 2% default | Maximum acceptable price movement |

## Market Types

<AccordionGroup>
  <Accordion title="Binary Markets" icon="toggle-on">
    **Yes/No Outcomes**
    
    - Two possible outcomes
    - Probabilities sum to 100%
    - Most common market type
    
    Example: "Will [Team] win the championship?"
  </Accordion>
  <Accordion title="Multi-Outcome Markets" icon="list-ol">
    **Multiple Possible Results**
    
    - 3+ possible outcomes
    - Probabilities sum to 100%
    - Higher complexity
    
    Example: "Who will win Best Picture?"
  </Accordion>
  <Accordion title="Scalar Markets" icon="chart-line">
    **Numeric Range Outcomes**
    
    - Predict within a numeric range
    - Payout scaled by accuracy
    - Used for quantities/prices
    
    Example: "What will be the final score?"
  </Accordion>
</AccordionGroup>

## Odds Configuration

<Tabs>
  <Tab title="Odds Display">
    | Format | Example |
    |--------|---------|
    | Percentage | 65% |
    | Decimal | 1.54 |
    | Implied | 65 cents per dollar |
    
    <Info>
      Versus displays odds as percentages by default. You can change this in Settings.
    </Info>
  </Tab>
  <Tab title="Odds Movement">
    | Movement Type | Cause |
    |---------------|-------|
    | Small (< 1%) | Normal trading flow |
    | Medium (1-5%) | Moderate volume or news |
    | Large (> 5%) | High volume or major news |
    
    Movement is instantaneous after each prediction.
  </Tab>

## Ranking Formula

```
Versus Score (VS) = (Weighted Win Rate × Volume Factor × Consistency Bonus) - Risk Penalty
```

| Component | Weight |
|-----------|--------|
| Base Win Rate | 40% |
| Volume Factor | 20% |
| Consistency Bonus | 25% |
| Risk Penalty | -15% max |

## Fee Schedule

| Fee Type | Amount | When Applied |
|----------|--------|--------------|
| Platform fee | 2% of profit | On winning predictions |
| Challenge fee | 0% | Never |
| League entry fee | Varies | Set by league creator |
| Withdrawal fee | Network cost | On token withdrawal |

<Note>
  Losing predictions incur no additional fees. You only lose your prediction amount.
</Note>

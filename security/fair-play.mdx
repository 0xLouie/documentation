# Fair Play Principles

Versus is built on fair competition. Our systems and policies ensure everyone competes on equal footing.

---

<Note>

</Note>

## Prohibited Activities

<AccordionGroup>
  <Accordion title="Multiple Accounts" icon="users-slash">
    **One Person, One Account**
    
    Prohibited:
    - Creating multiple accounts
    - Using others' accounts
    - Account sharing
    - Selling/buying accounts
    
    Detection:
    - Device fingerprinting
    - IP analysis
    - Behavioral patterns
    - Verification requirements
    
    Consequences:
    - All accounts banned
    - Funds frozen
    - Leaderboard removal
    - Permanent platform ban
  </Accordion>
  <Accordion title="Collusion" icon="handshake-slash">
    **No Coordinated Manipulation**
    
    Prohibited:
    - Coordinating positions with others
    - Artificially moving odds
    - Challenge manipulation
    - League fixing
    
    Detection:
    - Pattern analysis
    - Network mapping
    - Statistical anomaly detection
    - User reports
    
    Consequences:
    - Account suspension
    - Position reversal
    - Prize forfeiture
    - Permanent ban
  </Accordion>
  <Accordion title="Market Manipulation" icon="chart-line">
    **No Gaming the System**
    
    Prohibited:
    - Wash trading (self-dealing)
    - Spoofing positions
    - Information manipulation
    - Exploitation of bugs
    
    Detection:
    - Trade pattern analysis
    - Anomaly detection
    - AMM monitoring
    - Automated flags
    
    Consequences:
    - Trades reversed
    - Profits seized
    - Account terminated
    - Legal action possible
  </Accordion>
  <Accordion title="Automation & Bots" icon="robot">
    **No Unauthorized Automation**
    
    Prohibited:
    - Trading bots without approval
    - Automated account actions
    - API abuse
    - Scraping for advantage
    
    Detection:
    - Request pattern analysis
    - Rate limiting
    - CAPTCHA challenges
    - Behavioral analysis
    
    Note: API access available for approved use cases.
  </Accordion>
</AccordionGroup>

## Detection Systems

We employ multiple layers of protection:

| System | Purpose |
|--------|---------|
| **ML Models** | Identify suspicious patterns |
| **Network Analysis** | Detect related accounts |
| **Behavioral Fingerprinting** | Recognize repeat offenders |
| **Statistical Monitors** | Flag anomalous activity |
| **Community Reports** | User-submitted concerns |

<Note>
  Detection systems run continuously. Sophisticated manipulation will be caught.
</Note>

---

## Investigation Process

When issues are detected:

<Steps>
  <Step title="Flag">
    System or user flags suspicious activity for review.
  </Step>
  <Step title="Investigation">
    Team reviews evidence, patterns, and context.
  </Step>
  <Step title="Determination">
    Decision made based on evidence and policy.
  </Step>
  <Step title="Action">
    Appropriate consequence applied (warning to ban).
  </Step>
  <Step title="Appeal">
    User may appeal within 30 days with new evidence.
  </Step>
</Steps>

## Consequences Ladder

| Offense Level | First Offense | Repeat Offense |
|---------------|---------------|----------------|
| **Minor** | Warning | 7-day suspension |
| **Moderate** | 7-day suspension | 30-day ban |
| **Major** | 30-day ban | Permanent ban |
| **Severe** | Permanent ban | Legal action |

## Fair Play Rewards

We recognize players who exemplify fair competition:

- **Fair Play Badge**: For accounts with zero violations
- **Community Guardian**: For users who help maintain integrity
- **Seasonal Recognition**: Highlights in fair play reports

---

## Appeals Process

If you believe action was taken in error:

1. Contact fairplay@versusmarkets.io within 30 days
2. Provide evidence supporting your case
3. Team reviews appeal (5-10 business days)
4. Final decision communicated via email
5. Decision is binding

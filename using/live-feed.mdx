## Feed Item Types

Each item in your feed shows:

<AccordionGroup>
  <Accordion title="Prediction Entry" icon="crosshairs">
    When someone enters a new position:
    
    - User avatar and name
    - Market name and outcome selected
    - Entry odds
    - Position size (if public)
    - Time of entry
    
    **Actions**: Challenge, Copy Position, React
  </Accordion>
  <Accordion title="Position Resolved" icon="flag-checkered">
    When a prediction closes:
    
    - User avatar and name
    - Market name and outcome
    - Result (Win/Loss)
    - PnL amount
    - Celebratory animation for big wins ğŸ‰
    
    **Actions**: Congratulate, Challenge to Rematch, View Details
  </Accordion>
  <Accordion title="Challenge Update" icon="swords">
    Head-to-head activity:
    
    - Challenger and challenged
    - Market and stakes
    - Status (Pending, Accepted, Resolved)
    - Winner announcement
    
    **Actions**: Accept (if challenged), Watch, Create Similar Challenge
  </Accordion>
  <Accordion title="Streak Alert" icon="fire">
    Notable winning or losing streaks:
    
    - User and streak count
    - Recent predictions in streak
    - Streak type (daily, weekly, category)
    
    **Actions**: Follow, Challenge, View Profile
  </Accordion>
</AccordionGroup>

## Feed Controls

### Filtering

| Filter | Effect |
|--------|--------|
| **Category** | Show only specific topics |
| **Friends Only** | Hide non-friend activity |
| **Big Moves** | Only significant PnL events |
| **Challenges** | Focus on head-to-head action |

## Real-Time Updates

The feed updates automatically:

- **WebSocket Connection**: Instant updates without refresh
- **Smart Batching**: Multiple events grouped for readability
- **Offline Queue**: Catches you up when you return
- **Pull to Refresh**: Manual refresh always available

<Note>
  On mobile, the feed conserves battery by reducing update frequency when the app is backgrounded.
</Note>

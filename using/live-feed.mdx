# Live Feed

**Location:** `/terminal/live-feed`

The Live Feed keeps you connected to what's happening across your network. See predictions, wins, and losses as they happen—in real-time.

---

## Activity Stream

The feed updates continuously with actions from your network, showing you:

<CardGroup cols={3}>
  <Card title="New Predictions" icon="crosshairs">
    See when friends enter new markets
  </Card>
  <Card title="Betting Wins" icon="trophy">
    Celebrate victories with your network
  </Card>
  <Card title="Betting Losses" icon="circle-xmark">
    Commiserate (or gloat) over losses
  </Card>
</CardGroup>

---

## Feed Items

Each activity in your feed shows:

| Element | Description |
|---------|-------------|
| **User** | Who took the action |
| **Action** | What they did (predicted, won, lost) |
| **Market** | Which market it's about |
| **Details** | Side taken, amount, outcome |
| **Time** | When it happened |

<AccordionGroup>

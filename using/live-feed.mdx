---

## Activity Stream

The feed updates continuously with actions from your network:

  <Card title="New Predictions" icon="crosshairs">

  </Card>
  <Card title="Betting Wins" icon="trophy">
    Celebrate victories with your network
  </Card>
  <Card title="Betting Losses" icon="circle-xmark">
    Commiserate (or gloat) over losses
  </Card>
</CardGroup>

---

## Feed Items

Each activity shows:

| Element | Description |
|---------|-------------|
| **User** | Who took the action |
| **Action** | What they did (predicted, won, lost) |
| **Market** | Which market it's about |
| **Details** | Side taken, amount, outcome |
| **Time** | When it happened |

## Leaderboard Views

<Tabs>
  <Tab title="Global Rankings">
    **Everyone on Versus**
    
    The definitive ranking of all users based on performance.
    
    <Info>
      Global rankings update in real-time as predictions resolve.
    </Info>
  </Tab>
  <Tab title="Friends Rankings">
    **Your Network Only**
    
    A filtered view showing only your connected friends. This is where the real competition happens.
    
    <Tip>
      Connect more friends to make this leaderboard more competitive.
    </Tip>
  </Tab>
</Tabs>

## Ranking Metrics

The leaderboard shows detailed performance data:

| Metric | Description |
|--------|-------------|
| **Rank** | Position on the leaderboard |
| **User Name** | Profile name and avatar |
| **Total Predictions** | Number of predictions made |
| **Win Rate** | Percentage of correct predictions |
| **Avg Trade Price** | Average entry odds |
| **Total Profit** | Cumulative earnings |

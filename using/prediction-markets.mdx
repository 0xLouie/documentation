## Market Information

Each market card displays:

| Element | Description |
|---------|-------------|
| **Title** | Clear description of the event |
| **Category** | Topic classification with icon |
| **Resolution Date** | When the outcome is determined |
| **Current Odds** | Live probability for each outcome |
| **Volume** | Total amount predicted on this market |
| **Participants** | Number of unique predictors |
| **Friend Activity** | How many friends have positions |

## Filtering Markets

<AccordionGroup>
  <Accordion title="By Category" icon="folder">
    Select one or more categories to narrow your view:
    - Sports, Politics, Entertainment, Finance, Crypto, Culture
    - Sub-categories available within each
  </Accordion>
  <Accordion title="By Time" icon="clock">
    Filter by resolution window:
    - Resolving Today
    - This Week
    - This Month
    - 3+ Months Out
  </Accordion>
  <Accordion title="By Volume" icon="chart-bar">
    Sort by market activity:
    - Highest Volume
    - Trending (fastest growth)
    - New Markets
    - Lowest Volume (contrarian)
  </Accordion>
  <Accordion title="By Social" icon="users">
    Focus on network activity:
    - Friends Active
    - Following Active
    - Most Challenged
    - Controversial (split odds)
  </Accordion>
</AccordionGroup>

## Market Status

Markets go through several stages:

<Steps>
  <Step title="Open">
    Active trading, predictions accepted, odds moving
  </Step>
  <Step title="Locked">
    Event in progress, no new predictions, existing positions locked
  </Step>
  <Step title="Resolving">
    Outcome determined, awaiting official confirmation
  </Step>
  <Step title="Resolved">
    Final result posted, PnL distributed to winners
  </Step>
</Steps>
